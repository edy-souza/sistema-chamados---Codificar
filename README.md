📌 Sistema de Chamados Internos
📖 Descrição

Sistema simples para gerenciamento de chamados internos de suporte, permitindo criação, listagem e atribuição de responsáveis.

🚀 Tecnologias utilizadas
HTML5
CSS3
JavaScript (Vanilla)
LocalStorage (simulação de banco de dados)

⚙️ Como executar o projeto
Baixar ou clonar o repositório:
git clone URL_DO_REPOSITORIO
Abrir o arquivo:
index.html
Executar diretamente no navegador (não requer servidor)

🧠 Decisões técnicas
Uso de LocalStorage para persistência de dados, evitando necessidade de backend.
Responsáveis definidos estaticamente no código para simplificação.
Sistema construído em JavaScript puro para facilitar execução e avaliação.
Status "fechado" utilizado para controle de chamados em aberto.

📌 Funcionalidades
Criar chamados
Listar chamados
Atribuição de responsável manual
Atribuição automática baseada na carga de trabalho
Persistência local dos dados
